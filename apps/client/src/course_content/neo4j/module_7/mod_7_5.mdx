## Your Graph Cheat Sheet

| Feature         | Cypher Command                           | Purpose               |
| :-------------- | :--------------------------------------- | :-------------------- |
| **Create Node** | `CREATE (n:Label {key: 'val'})`          | Add data.             |
| **Connect**     | `MATCH (a), (b) CREATE (a)-[:TYPE]->(b)` | Add relationships.    |
| **Logic**       | `MERGE (n:User {id: 1})`                 | Create if not exists. |
| **Filter**      | `WHERE n.age > 21`                       | Refine results.       |
| **Stats**       | `RETURN count(*)`                        | Aggregation.          |
| **Algorithms**  | `CALL gds.pageRank.stream(...)`          | Graph Analysis.       |

---

## Final Wisdom

- Use **MongoDB** for flexible documents.
